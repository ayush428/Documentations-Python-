Variables

In Python, variables are used to store data values. Variables do not need explicit declaration to be created. A variable is created the moment you assign a value to it.

Python is dynamically typed, meaning you don't need to declare the type of a variable explicitly. The type is inferred based on the value.

Variable Naming Conventions

  • Variable names must start with a letter or an underscore (_).
  • Variable names cannot start with a number.
  • Variable names are case-sensitive (e.g., name and Name are different).
  • Python has reserved keywords that cannot be used as variable names (e.g., if, while, return).
x = 10
y = 3.14
name = "Python"
# Printing variables
print(x)
print(y)
print(name)

Multiple Assignments

You can assign multiple variables in one line.

x, y, z = 1, 2, 3
a = b = c = 0

Use the type() function to check the type of a variable.

Global and Local Variables

In Python, variables defined inside a function are local to that function, while variables defined outside of a function are global. To modify a global variable inside a function, use the global keyword.

Data Types

Python supports various data types. Here are some of them:

  1. Numeric Types:
    • Integer: x = 10
    • Float: pi = 3.14
    • Complex: z = 2 + 3j
  2. Sequence Types:
    • String: greeting = 'Hello'
    • List: numbers = [1, 2, 3]
    • Tuple: coordinates = (10, 20)
    • Range: r = range(5)
  3. Mapping Types:
    • Dictionary: person = {"name": "Alice", "age": 25}
  4. Set Types:
    • Set: unique_numbers = {1, 2, 3}
    • Frozen Set: frozen = frozenset([1, 2, 3])
  5. Boolean Types:
    • flag = True
  6. Binary Types:
    • Bytes: data = b"hello"
    • Bytearray: data = bytearray(5)
    • Memoryview: data = memoryview(b"hello")
  7. None Type: x = None
If...else Statement

Python uses the if, elif, and else statements for conditional logic.

x = 10
if x > 10:
print("x is greater than 10")
elif x == 10:
print("x is equal to 10")
else:
print("x is less than 10")

Ternary if-else in Python:

result = "x is greater than 10" if x > 10 else "x is 10 or less"
print(result)
Loops

Python supports while loops and for loops for iteration.

count = 0
while count < 5:
print("Count is:", count)
count += 1

Infinite loop:

while True:
print("This is an infinite loop!")

Use break to exit a loop prematurely:

count = 0
while True:
if count >= 5:
break
print("Count is:", count)
count += 1

Use continue to skip the rest of the current iteration:

count = 0
while count < 5:
count += 1
if count == 3:
continue
print("Count is:", count)

Static method:

class Utils:
@staticmethod
def multiply(a, b):
return a * b
result = Utils.multiply(5, 6)
