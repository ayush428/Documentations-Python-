Variables

In Python, variables are used to store data values, like any other language. Variables do not need explicit declaration to be created. A variable is created the moment you assign a value to it.

Python is dynamically typed, meaning you don't need to declare the type of a variable explicitly. The type is inferred based on the value.

Variable Naming conventions

  • Variable names must start with a letter or an underscore _.
  • ariable names cannot start with a number.
  • Variable names are case-sensitive (name and Name are different).
  • Python has reserved keywords that cannot be used as variable names (e.g., if, while, return , etc.).
x = 10
y = 3.14
name = "Python"
# Printing variables
print(x)
print(y)
print(name)
